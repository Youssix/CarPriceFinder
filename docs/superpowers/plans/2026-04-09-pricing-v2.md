# Pricing v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor Carlytics pricing from 3 plans (Starter / Pro / Agency) to a single Pro plan at 89€/mois, with a "first reveal" free hook on the extension.

**Architecture:** Extension + landing + dashboard + server + docs. Changes are:
1. **Marketing surface** (landing, dashboard Upgrade page, docs) — strip 2 of 3 plans, strip padding features from the pitch.
2. **Server** (`stripe.js`) — remove references to non-existent Starter/Agency price IDs from the checkout map.
3. **Extension** (`intercept.js`, `bca-intercept.js`) — add first-reveal logic: on the very first analysis after install, show the chiffres even for non-paid users; persist a `firstRevealUsed` flag in `chrome.storage.local` via the existing content-bridge pattern.

No DB migration (0 paying customers). No new tests — project has no test suite. Verification is manual via Vite preview for dashboard and checklist for extension/landing.

**Tech Stack:** Vanilla JS (extension), React + Vite (dashboard), static HTML (landing), Node/Express (server), Stripe, Chrome Extension Manifest V3.

  // --- First reveal (pricing v2) ---
  // Lit/ecrit le flag firstRevealUsed via le bridge storage (meme mecanique
  // que loadExtensionSettings). Utilise en flow : si l'user n'est pas payant
  // et n'a pas encore consomme son reveal, on affiche les chiffres UNE fois
  // et on set le flag immediatement.
  function getFirstRevealUsed() {
    return new Promise((resolve) => {
      const requestId = 'fru_get_' + Math.random().toString(36).substr(2, 9);
      const timer = setTimeout(() => {
        window.removeEventListener('message', handler);
        resolve(false); // fail-open : si le bridge ne repond pas, on considere non-consomme
      }, 2000);

      function handler(event) {
        if (event.source !== window) return;
        const msg = event.data;
        if (!msg || msg.type !== 'STORAGE_RESPONSE' || msg.requestId !== requestId) return;
        clearTimeout(timer);
        window.removeEventListener('message', handler);
        resolve(msg.data && msg.data.firstRevealUsed === true);
      }

      window.addEventListener('message', handler);
      window.postMessage({
        type: 'STORAGE_REQUEST',
        action: 'get',
        keys: ['firstRevealUsed'],
        requestId
      }, '*');
    });
  }

  function setFirstRevealUsed() {
    const requestId = 'fru_set_' + Math.random().toString(36).substr(2, 9);
    window.postMessage({
      type: 'STORAGE_REQUEST',
      action: 'set',
      data: { firstRevealUsed: true },
      requestId
    }, '*');
    // Fire-and-forget : on n'attend pas la confirmation, le flag sera set
    // avant le prochain render.
  }

  function getFirstRevealUsed() {
    return new Promise((resolve) => {
      const requestId = 'bca_fru_get_' + Math.random().toString(36).substr(2, 9);
      const timer = setTimeout(() => {
        window.removeEventListener('message', handler);
        resolve(false);
      }, 2000);

      function handler(event) {
        if (event.source !== window) return;
        const msg = event.data;
        if (!msg || msg.type !== 'STORAGE_RESPONSE' || msg.requestId !== requestId) return;
        clearTimeout(timer);
        window.removeEventListener('message', handler);
        resolve(msg.data && msg.data.firstRevealUsed === true);
      }

      window.addEventListener('message', handler);
      window.postMessage({
        type: 'STORAGE_REQUEST',
        action: 'get',
        keys: ['firstRevealUsed'],
        requestId
      }, '*');
    });
