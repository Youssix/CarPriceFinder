# Extension Icons

Since I can't create actual image files, here are the icon specifications for your extension:

## Required Icons

Create these icons and place them in the root directory:

### icon16.png (16x16 pixels)
- Small icon for extension toolbar
- Simple car + AI symbol
- Colors: Blue (#007bff) and green (#28a745)

### icon48.png (48x48 pixels)  
- Medium icon for extension management
- Car silhouette with "AI" badge
- Clear, readable design

### icon128.png (128x128 pixels)
- Large icon for Chrome Web Store
- Detailed car + price tag + AI brain
- Professional, modern design

## Design Suggestions

**Concept**: A car silhouette with an AI/brain icon and price tag
**Colors**: 
- Primary: #007bff (blue)
- Secondary: #28a745 (green) 
- Accent: #ffc107 (yellow/gold)

**Elements**:
- Car outline (simple, recognizable)
- AI symbol (brain, circuit, or "AI" text)
- Price tag or euro symbol
- Modern, flat design style

## Add to manifest.json

```json
"icons": {
  "16": "icon16.png",
  "48": "icon48.png", 
  "128": "icon128.png"
}
```

You can create these icons using:
- Figma (free, web-based)
- Canva (templates available)
- GIMP (free desktop app)
- Adobe Illustrator
- Online icon generators

## Quick Icon Creation

1. Search for "car icon generator" online
2. Use a combination of:
   - Car silhouette
   - AI/tech elements
   - Price/money symbols
3. Export in required sizes
4. Ensure good contrast and readability
