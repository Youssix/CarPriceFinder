# Guide d'installation — Carlytics Beta

## Ce dont vous avez besoin
- Google Chrome (ordinateur, pas telephone)
- Le fichier ZIP de l'extension (envoye par email/WhatsApp)
- Votre cle API (fournie par l'equipe Carlytics)

---

## Etape 1 : Telecharger et decompresser

Telechargez le fichier `carlytics-extension-beta.zip` et decompressez-le dans un dossier sur votre ordinateur (ex: Bureau/Carlytics).

> **Important** : gardez ce dossier, ne le supprimez pas — Chrome en a besoin.

---

## Etape 2 : Ouvrir les extensions Chrome

1. Ouvrez Chrome
2. Tapez dans la barre d'adresse : `chrome://extensions/`
3. Activez le **Mode developpeur** (interrupteur en haut a droite)

---

## Etape 3 : Charger l'extension

1. Cliquez sur **"Charger l'extension non empaquetee"** (bouton en haut a gauche)
2. Selectionnez le dossier que vous avez decompresse a l'etape 1
3. L'extension **CarPriceFinder AI** apparait dans la liste

---

## Etape 4 : Configurer votre cle API

1. Cliquez sur l'icone de l'extension (puzzle en haut a droite de Chrome, puis "CarPriceFinder AI")
2. Dans le popup, entrez votre **cle API** dans le champ prevu
3. Cliquez sur **Sauvegarder**

> Votre cle API ressemble a : `cpf_live_abc123def456...`
> Elle vous sera envoyee par l'equipe Carlytics.

---

## Etape 5 : Utiliser sur Auto1

1. Allez sur **auto1.com** et connectez-vous
2. Naviguez vers un vehicule
3. L'extension affiche automatiquement :
   - **Prix LeBonCoin estime** (mediane du marche)
   - **Marge potentielle** (LBC - Auto1)
   - **Options premium detectees** (M-Sport, AMG, S-Line...)

> Le premier chargement peut prendre 2-3 secondes. Les vehicules deja analyses s'affichent instantanement grace au cache.

---

## Problemes courants

| Probleme | Solution |
|----------|----------|
| L'extension ne s'affiche pas sur Auto1 | Rechargez la page (F5). Verifiez que l'extension est activee dans chrome://extensions/ |
| "Cle API requise" | Ouvrez le popup de l'extension et entrez votre cle API |
| "Erreur serveur" | Verifiez votre connexion internet. Si le probleme persiste, contactez-nous |
| Les prix ne s'affichent pas | Attendez 2-3 secondes. Le serveur interroge LeBonCoin en temps reel |
